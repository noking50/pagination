# Pagination

List of pagination

## Installing

### install
```
composer required noking50/pagination
```

### config
```
return [
    'items' => 10,
    'pages' => 10,
    'data_display' => [
        10 => '10',
        25 => '25',
        50 => '50',
        100 => '100',
    ],
];
```
